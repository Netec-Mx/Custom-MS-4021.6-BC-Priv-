# Experiencia de inmersión en Copilot:Recursos Humanos



## Plataforma de laboratorios



Te damos la bienvenida a la **plataforma de laboratorios** del curso **Experiencia de inmersión en Copilot:Recursos Humanos**. Aquí podrás explorar diferentes tecnologías a través de prácticas guiadas. ¡Desarrolla tus habilidades y lleva tus conocimientos al siguiente nivel!



## Lista de laboratorios



Cada uno de estos laboratorios está diseñado para ofrecerte una experiencia práctica. Haz clic en los enlaces para comenzar.



### [Práctica 1. Preparar el contexto de talento humano desde Outlook](Capitulo_1/README.md) 

  - **Descripción**: En esta práctica asumirás el rol de un analista de Talento Humano encargado de gestionar y priorizar comunicaciones críticas relacionadas con procesos de selección y gestión del personal. Utilizando Microsoft Outlook y Microsoft Copilot, analizarás cadenas de correos electrónicos para identificar información relevante, resumir conversaciones, detectar riesgos, responsables y acciones pendientes. Finalmente, estructurarás un bloque de contexto que servirá como insumo para posteriores análisis en Microsoft Excel y Microsoft 365 Copilot Chat, facilitando la toma de decisiones y el seguimiento de los procesos de Talento Humano..

  - ⏱️ **Duración estimada**: 15 min.



### [Práctica 2. Analizar información de colaboradores y candidatos con Copilot en Excel](Capitulo_2/README.md)

  - **Descripción**: Partiendo de un conjunto de datos del área de Talento Humano, emplearás Microsoft Copilot en Excel para identificar patrones, evaluar indicadores clave y comparar competencias entre candidatos y colaboradores internos. Como resultado, generarás explicaciones claras y recomendaciones que faciliten la toma de decisiones en procesos de selección, desarrollo y movilidad del talento.

  - ⏱️ **Duración estimada**: 20 min.



### [Práctica 3. Preparar entrevistas y estrategias de desarrollo con Microsoft 365 Copilot Chat y Prompt Coach](Capitulo_3/README.md)

  - **Descripción**: A partir de un escenario del área de Talento Humano, emplearás Microsoft 365 Copilot Chat para crear y perfeccionar una descripción de cargo acorde con las responsabilidades del puesto. Posteriormente, prepararás entrevistas mediante simulaciones de juego de roles y preguntas orientadas por competencias. Finalmente, utilizarás Prompt Coach para diseñar estrategias de desarrollo y fortalecer la comunicación asertiva de los líderes durante sus interacciones con los colaboradores.

  - ⏱️**Duración estimada**: 20 min.



### [Práctica 4. Crear una presentación ejecutiva de Talento Humano con Copilot en PowerPoint](CHAPTER_04/ch04-smart-savings-goals/README.mdCapitulo_4/README.md
  - **Descripción**: Integrarás los resultados obtenidos durante las prácticas anteriores para elaborar una presentación ejecutiva con Microsoft Copilot en PowerPoint. A partir de los hallazgos relacionados con análisis de talento, competencias, perfiles, riesgos de rotación y movilidad interna, estructurarás y perfeccionarás las diapositivas, construyendo una narrativa clara y orientada a la toma de decisiones por parte de líderes de Talento Humano y áreas directivas..

  - ⏱️**Duración estimada**: 20 min.
